# Sistem Pelaporan Prestasi Mahasiswa -- UAS

Backend API untuk sistem pencatatan dan manajemen prestasi mahasiswa.
Dibangun menggunakan Golang + Fiber + PostgreSQL.

## Fitur
- Auth JWT (Login, Register)
- Manajemen User & Role
- Create & Update Draft Prestasi
- Swagger API Documentation
- Unit Testing

## Tech Stack
- Go
- Fiber
- PostgreSQL
- Swagger (swaggo)
